启动 @cdl-pkg/main-application 编辑器

```shell
pnpm dev --filter @cdl-pkg/main-application
```

启动 @cdl-pkg/main-server 通用服务

```shell
pnpm dev --filter @cdl-pkg/main-server -- -p 3001
```
